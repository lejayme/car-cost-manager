## 📖 项目简介

**CarCost** 是一款帮助用户清晰掌握车辆使用开销的工具，支持记录充电费、停车费、加油费、保险、保养等各种用车支出，并提供详细的数据分析和统计报表。

### ✨ 核心功能

- 📝 **快速记账** - 记录每笔用车开销，支持分类、备注、拍照
- 📊 **数据分析** - 日均成本、月度汇总、分类占比、趋势图表
- 🚗 **多车管理** - 支持多辆车独立记录，切换查看
- 🔄 **云端同步** - 数据实时同步，多端无缝切换
- 📤 **导出报表** - 支持 Excel/PDF 格式导出

## 🎯 适用人群
